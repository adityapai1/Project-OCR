## Installation Guide

Clone the project : 

```bash
  git clone https://github.com/adityapai1/Easy_OCR.git
```

Install Flask , Pytorch  : 

```bash
pip install Flask 
pip install pytorch
pip install pandas

```
Install easyocr : 

```bash
pip install easyocr
```
