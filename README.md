## Installation Guide

Clone the project : 

```bash
  git clone https://github.com/adityapai1/Easy_OCR.git
```


Install easyocr : 

```bash
pip install easyocr
```


Install Flask : 

```bash
pip install Flask
```

Install Pillow : 

```bash
pip install Pillow
```

Execute the python script : 

* Works with **python 3.9** only

```bash
  python application.py
```
